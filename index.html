Cursed Revolver

Cursed Revolver is a 2d game developed in JavaScript and Html. It features real time combat, multiple enemy types, and a boss fight. Developed by a team of 4. Voted #1 game in the class out of 25 others.

Link to Game GitHub Repository

Radical Racer

Radical Racer is a 3d racing game developed with Python and the Ursina Engine. Features 3d graphics and competitive race timer.

Download Exe File GitHub Repository

Understone

Understone is a 2d game based on the video game Deep Rock Galactic. Fight through hordes of bug to get to the alien egg. It features real time combat, many different weapons, and multiple enemy types. Developed by a small team in Java.

Download Jar File GitHub Repository

Mine Sweeper

Browser based game of mine sweeper. Includes animations for winning and losing. Uses bootstrap themes and JavaScript scripting with Jquery. Hosted on GitHub Pages.

Website Link GitHub Repository

Enchanted Emporium

Browser based online storefront for wizards. Stores many products with images, descriptions, tags, and reviews. Search page provides many different searching and ordering options. Uses an Html and Jquery frontend connecting to a Node.js backend using MySQL database.

GitHub Repository

Gloop Game

2d game developed in Java using Java Swing. Includes item crafting, inventory management, item progression, and buildable structures.

Download Jar File GitHub Repository

Chess Simulator

Chess simulator developed in Java using Java Swing. Allows two players to play against each other. Includes real time chess rules, piece movement, and checkmate and stalemate detection.

Download Jar File GitHub Repository
